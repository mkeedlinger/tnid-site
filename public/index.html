TNID

Type-safe, named, unique identifiers

UUID-compatible IDs with embedded type names. Human-readable, compile-time type safety, zero ambiguity.

Type-safe

Different ID types are incompatible at compile time. Pass a UserId where a PostId is expected? Compiler error.

Human-readable

IDs are self-documenting. See user.Br2flcNDfF6LYICnT in a log and know exactly what it is.

UUID-compatible

Store in any UUID column. Every TNID is a valid UUIDv8. Use existing UUID tooling and infrastructure.
